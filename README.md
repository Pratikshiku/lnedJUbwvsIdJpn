## Online Teaching Platform System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/152571c40f57494fa3c1bc0ea2da1b7f/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Learning Materials (Browse, Download, Comment, Favorite)
5. Communication Forum
6. Exam Papers List, Exams
7. Announcement Information
8. Message Feedback
9. Personal Center
10. My Publications
11. Exam Records
12. Wrong Answer Book
13. My Favorites
14. Change Password
15. Personal Information
16. My Favorites Management
17. Message Board Management
18. Exam Management


**Administrator Functions:**
1. Login
2. Change Password
3. Student Management
4. Material Type Management
5. Learning Material Management
6. Communication Forum
7. My Favorites Management
8. Exam Paper Management
9. Message Board Management
10. Question Management
11. Carousel Management
12. Announcement Information Management
13. Exam Management
